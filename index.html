📡 SignalRadar

v3.0 Zero Tracking Made in 🇩🇪 Open Source

Mobilfunk-Abdeckungskarte für Deutschland. Signalstärke-Berechnung via COST 231 Hata-Modell mit echten Sendemasten-Daten.

Features:
🔍 Ort, PLZ oder Adresse suchen
📍 GPS-Standort mit Live-Tracking
🛰️ Satelliten- oder Straßenkarte
🌡️ Signal-Heatmap (Zoom 12+)
⚡ Integrierter Speed Test
⚖️ Provider-Vergleich an deinem Standort
📏 Entfernungen messen

Netze: O2/1&1, Telekom, Vodafone

Datenquellen: OpenStreetMap, Overpass API

Entwickelt von PurpleDoubleD
Keine Daten werden gesammelt oder weitergegeben.
Deine Privatsphäre ist uns wichtig.
